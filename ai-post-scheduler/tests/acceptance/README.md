# Phase 3: Live XAMPP Acceptance Tests

This directory contains acceptance tests for testing the AI Post Scheduler plugin against a live WordPress instance running on your XAMPP server.

## Prerequisites

### 1. XAMPP Setup
- WordPress running at: `http://localhost/wp_nunezscheduler/`
- Admin user: `admin` / `admin`
- AI Post Scheduler plugin: activated and configured
- Meow Apps AI Engine: connected to Gemini API (real API calls)

### 2. Chrome/ChromeDriver
For WebDriver tests to run, you need:
- Google Chrome installed
- ChromeDriver downloaded (matching your Chrome version)
- OR: Use Selenium Docker container

### 3. Selenium Server

#### Option A: Using Docker (Recommended)
```bash
docker run -d -p 4444:4444 --name selenium selenium/standalone-chrome
```

#### Option B: Download and Run Standalone
1. Download Selenium Standalone Server
2. Download ChromeDriver matching your Chrome version
3. Run Selenium:
```bash
java -Dwebdriver.chrome.driver=/path/to/chromedriver -jar selenium-server-standalone.jar
```

## Running the Tests

### Prerequisites: Start Selenium
Before running tests, ensure Selenium is running on localhost:4444

#### Using Docker:
```bash
docker run -d -p 4444:4444 --name selenium selenium/standalone-chrome
```

#### Check if Selenium is running:
```bash
curl http://localhost:4444/wd/hub/status
```

### Run All Acceptance Tests
```bash
cd ai-post-scheduler
php vendor/bin/codecept run acceptance
```

### Run Specific Test File
```bash
php vendor/bin/codecept run acceptance AdminTemplateScheduleWorkflowCest
```

### Run Specific Test
```bash
php vendor/bin/codecept run acceptance AdminTemplateScheduleWorkflowCest:adminLoginTest
```

### Run with Verbose Output
```bash
php vendor/bin/codecept run acceptance -vvv
```

### Run with Screenshots on Failure
```bash
php vendor/bin/codecept run acceptance --steps
```

## Configuration

### codeception.yml
The main configuration file sets up:
- Base URL: `http://localhost/wp_nunezscheduler/`
- Browser: Chrome
- WebDriver host: localhost
- WebDriver port: 4444

To modify:
```yaml
suites:
    Acceptance:
        modules:
            enabled:
                - WebDriver:
                    url: http://localhost/wp_nunezscheduler/
                    browser: chrome
```

### AcceptanceHelper
Located in `tests/Support/AcceptanceHelper.php`, provides utilities:
- `loginToWordPress($username, $password)` - Login helper
- `navigateToAdminPage($page)` - Navigate to plugin pages
- `isLoggedIn()` - Check login status
- `waitForAjax()` - Wait for AJAX completion
- `clickByText($text)` - Click elements by text

## Creating New Tests

### Basic Test Structure
```php
class MyFeatureCest {
    public function myTestFunction(AcceptanceTester $I) {
        // Login
        $I->amOnPage('/wp-admin/');
        $I->fillField('user_login', 'admin');
        $I->fillField('user_pass', 'admin');
        $I->click('Log In');
        $I->waitForElementVisible('.wp-admin', 5);
        
        // Your test actions
        $I->see('Dashboard');
    }
}
```

### Using Helper Methods
```php
class MyFeatureCest {
    public function myTestFunction(AcceptanceTester $I, \Tests\Support\AcceptanceHelper $helper) {
        $helper->loginToWordPress('admin', 'admin');
        $helper->navigateToAdminPage('aips-templates');
        $helper->seePageTitle('Templates');
    }
}
```

## Troubleshooting

### Selenium Connection Failed
```
WebDriver connection refused at http://localhost:4444
```
**Solution**: Ensure Selenium is running on port 4444. Check with `curl http://localhost:4444/wd/hub/status`

### Tests Hang
```
Timeout waiting for element
```
**Solution**: Increase timeouts in test or check that XAMPP is running and site is accessible

### Login Fails
```
User login element not found
```
**Solution**: Verify WordPress is running at `http://localhost/wp_nunezscheduler/` and login page is accessible

### Chrome Not Found
```
WebDriver: Chrome not found
```
**Solution**: 
- Verify Chrome is installed
- Check Chrome binary path matches your system
- Use headless mode: add `--headless` to chromeOptions
