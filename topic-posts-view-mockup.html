4. Technical Implementation

Backend (PHP)

  • New AJAX Endpoint: aips_get_topic_posts
  • Modified Endpoint: aips_get_author_topics now includes post counts
  • Database Query: Joins author_topic_logs with WordPress posts

Frontend (JavaScript)

  • New Functions:
    • viewTopicPosts() - Handle badge click
    • loadTopicPosts() - Fetch posts via AJAX
    • renderTopicPosts() - Render posts table

  • Modal layout with large width for better readability
  • Responsive table design
