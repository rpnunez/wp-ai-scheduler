## 2025-04-26 - Template Threw Error when Passing History Class as Object\n**Learning:** Passing an instance of `AIPS_History` (instead of an array) to `history.php` caused a fatal error when trying to access array offsets.\n**Action:** Replaced `isset($history['items'])` with `isset($history['items']) && is_array($history)`, and retrieved paginated records by calling `$history->get_history()` if `$history` is an object and contains that method. This guarantees variables match what `history.php` expects and restores backward compatibility.
