## 2025-02-14 - Empty State Action Polish
**Learning:** In list views powered by PHP that support form filtering (e.g. `post-review.php`), an active filter yielding zero results requires its own specific empty state with an actionable "Clear Filters" button, otherwise users are shown a generic empty state (like "No Draft Posts") that incorrectly implies there are simply no posts *overall* rather than just *no results for their search*.
**Action:** When working on lists or tables, conditionally check if search parameters are active (`!empty($search_query) || !empty($template_id)`) to show a "No Results Found" empty state with a "Clear Filters" button instead of the default generic "No Items Yet" empty state.
