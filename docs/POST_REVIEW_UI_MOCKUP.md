## User Interactions

### Select Multiple Posts
1. Click checkboxes next to posts you want to act on
2. Click "Select All" to select all visible posts
3. Choose action from "Bulk Actions" dropdown
4. Click "Apply"
5. Confirm the action in popup dialog

### Publish a Post
1. Click [Publish] button on a post row
2. Confirm action in dialog: "Are you sure you want to publish this post?"
3. Post is published and removed from the list
4. Success notification appears

### View Logs
1. Click [View Logs] button on a post row
2. Modal opens showing:
   - The AI prompt used
   - Generation log details
   - Timestamps
3. Click [Close] or [×] to close modal

### Re-generate a Post
1. Click [Re-generate] button
2. Confirm: "Are you sure you want to regenerate this post? The current post will be deleted."
3. Current post is deleted
4. New generation starts
5. Post is removed from review queue
6. Success notification appears

### Delete a Post
1. Click [Delete] button
2. Confirm: "Are you sure you want to delete this post? This action cannot be undone."
3. Post is permanently deleted
4. Success notification appears
