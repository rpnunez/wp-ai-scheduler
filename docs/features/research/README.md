# Research Features

## Overview

The Research features in AI Post Scheduler provide powerful tools for discovering, analyzing, and planning content based on trending topics and data-driven insights. These features help you create content that is timely, relevant, and optimized for engagement.

## Key Components

### 1. Trending Topics Research
Discover what's currently popular in your niche using AI analysis. This feature scans for trends, news, and seasonal factors to suggest high-potential blog post topics.

### 2. Trending Topics Library
A centralized repository for all your researched topics. Store, organize, filter, and manage topic ideas for future use.

### 3. Content Planner
A visual calendar and scheduling tool that allows you to map out your content strategy. Drag and drop topics, assign templates, and visualize your publishing schedule.

## Quick Start

1. Go to **AI Post Scheduler → Trending Topics**
2. Enter your niche (e.g., "Digital Marketing")
3. Click **Research** to generate ideas
4. Select topics and click **Schedule** to add them to your planner
