# Trending Topics Library

## Overview

The Trending Topics Library is your database of discovered content ideas. It stores every topic generated from your research sessions, allowing you to build a backlog of ideas to draw from over time.

## Features

### 1. Storage
All topics generated via the Research tool are automatically saved to the library.

### 2. Filtering & Search
- **By Niche**: View topics only for a specific niche.
- **By Score**: Filter for high-scoring topics (e.g., Score > 80).
- **By Freshness**: See only topics researched recently (e.g., last 7 days).
- **Search**: Find specific topics by keyword.

### 3. Management
- **Delete**: Remove topics that aren't relevant or have already been used.
- **Bulk Actions**: Select multiple topics to delete or schedule at once.

## Workflow

1. **Review**: Periodically review your library to find overlooked gems.
2. **Curate**: Delete low-quality or irrelevant topics to keep the library clean.
3. **Action**: Select topics to schedule directly from the library view.

## Data Retention

Topics remain in the library indefinitely unless manually deleted. However, "freshness" is a key factor in relevance, so older trending topics may become less valuable over time. Use the "Fresh Only" filter to focus on current opportunities.
